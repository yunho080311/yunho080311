# 02-1 숫자형 학습 소감

# 오늘 배운 내용
- 파이썬의 다양한 숫자형(정수, 실수, 컴퓨터식 지수표현법과 8진수 16진수) 표현법을 배움.
- 사직연산( + - * / ) 외에도 '//', '**', '%' 연산자의 사용법을 배움.

# 내용을 토대로 깨달은 점
- 처음에는 나누기는 '/'만 있는 줄 알았는데 몫을 구하는 '//'와 '%'가 있다는 것을 깨달음
- 숫자의 제곱을 나타내는 수가 ^인줄 알았는데 **가 있어서 놀랐음
- 16진수는 16이상으로 올라가면 계산이 안되고 8진수도 똑같이 8이상으로 올라가면 계산이 안됨

# 실천
- 나만 보는 것이 아니라 남도 본다는 심정으로 정리를 꼼꼼히 하고 주석을 다는 습관을 기르겠음.
- //연산자는 몫 %연산자는 나머지를 기억하겠음
